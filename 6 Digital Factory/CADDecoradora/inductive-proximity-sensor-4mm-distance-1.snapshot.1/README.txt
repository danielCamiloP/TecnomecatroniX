Inductive Proximity Sensor

== IDN ==

Proximity Induktif ini bekerja dengan cara mendeteksi barang berbahan metal dengan jarak deteksi 4mm. Output dari sensor ini berkarakteristik NPN NO.

Pemodelan gambar menggunakan jangka sorong untuk mendapatkan dimensi yang akurat. Karena sensor induktif ini dipasarkan dengan berbagai pabrikan, terkadang memiliki perbedaan panjang benda sensor, namun diameternya kurang lebih konsisten.

== ENG ==

This proximity sensor outputs Open Drain NPN signal. Sensor using inductive coil to detect metallic object within 4mm surrounding sensor.


Since I have acquired several of these inductive proximity sensors, I can safe to say most of them have similar dimension with only the length that have variance between each manufacturers, the diameter should be consistent. This drawing created based on measured dimension using Caliper (with correct thread and its pitch length).
